readme

crateland is built upon the crate package for deno (view the code on github).
below is a basic example of crate at work.

main.ts 
import { Crate, file } from "https://crate.land/mod";

let crate = new Crate();
crate.title = "Crate";
crate.url = await file("index.html");
crate.width = 720, crate.height = 480;
crate.run();

index.html 
<html>
  <main>hello from index.html!</main>
</html>

if you're on mac or linux, run the command below in your terminal to spawn a crate —

deno run -A --unstable main.ts

(or, if you want only the necessary flags —

deno run --unstable --allow-env --allow-read --allow-ffi crate.ts)


if you're on windows, run the below command to create a standalone executable —

deno run -A --unstable https://crate.land/compile main.ts


— a demo of the above command

zip it with index.html and distribute!


future

crateland will compile to linux and mac, and universally as a standalone executable. it will be customizable through a single html file — the title, icon, and content will all be modified through index.html. as of now, it will be deliverable on windows only. thanks to deno compile and crate, deno-based graphical applications are now possible.

check out the demo!

the executable + index.html zipped are ~23mb, which is due to deno compile. hopefully in the future this size will get smaller.
